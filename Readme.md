### 특징

1. Java와 달리 변수를 선언할 때, 자료형을 명시적으로 선언하지 않는다 

```java
// java 에서의 변수 선언
  int num = 10
  printf("int num = %d\n", num);
```

```python
# Python 에서의 변수 선언
  num = 10
  print(num)
```

2. print 구문시 

```python
# 첫번째 방법
name = "손흥민"
print('name = '+name) # 출력 결과  name = 손흥민

# 두번째 방법 콤마로 할 시 앞에 뛰어쓰기 한칸 생김
print('name = ',name) # 출력 결과  name =  손흥민 
```
